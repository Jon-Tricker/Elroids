Goods and Minerals.

There are 3 types of item that can be traded:

  • Minerals - Obtained by breaking up 'rocks'.
  • Trade Goods - Various products that can be traded. Each "Goods" object contains a variable number of object. If the goods are damaged some of the objects are destroyed.
  • Ship components - A sub class of 'Trade Goods' containing individual ship components. Each object contains a single component which may be in a damaged state.

Free floating Goods and Minerals

Goods and Minerals can end up floating in space:

  • Minerals - Are represented as metalic cylinders. Colour indicates the mineral type.
  • Trade Goods/Ship Components - Are represented bs brown, rectangular 'crates'. Who knows what's in it? ... until you get it on board.

Floating objects can be picked up by ramming them.

Floating objects a limited lifetime. If not picked up (or destroyed) they self-destruct.

Dumping cargo

If NOT docked goods/minerals can be 'dumped'. This will leave them floating in space.

This is useful if you end up carrying illegal goods or want to clear out space in the bay. You can't 'dump' in dock ... it would make a mess on the floor!
